Configure Dialog Feedback Demo

Purpose: This demo shows the visual feedback banner that appears when users apply or save configuration changes in FastWeather.

Key Features:

  • Auto-dismisses after 10 seconds
  • Manual close button (×)
  • WCAG 2.2 AA compliant (role="status", aria-live="polite")
  • Keyboard accessible
  • Supports reduced motion preferences
  • High contrast mode compatible

Implementation: The banner is created in showConfigFeedback(saved) function in app.js and styled in styles.css.
