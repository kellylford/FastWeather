Why use this?

If you're not seeing the latest city data (like new countries), the service worker might be serving old cached files. This tool will clear all caches and force a fresh reload.

Manual alternatives:
  • Chrome/Edge: Press Ctrl + Shift + Delete, select "Cached images and files", click Clear
  • Firefox: Press Ctrl + Shift + Delete, select "Cache", click Clear Now
  • Hard Refresh: Ctrl + Shift + R (or Cmd + Shift + R on Mac)
